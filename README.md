## Usage

### Basic Workflow
1. **Navigate** to any YouTube Video, Short, or Community Post
2. **Click** the extension icon in your browser toolbar
3. **Click "Auto-Scroll"** to load all comments (optional)
4. **Click "Extract Comments"** to gather all comments
5. **Apply Filters** to narrow down entries (optional)
6. **Pick Winners** with the dramatic slot machine reveal!
7. **Export** winners or all comments in your preferred format

### Advanced Tips
- Use the **Refresh** button (top-right) to re-apply filters
- Use **Clear Session** to start fresh
- Use **View All** to see all comments in a searchable modal
- Export comments before picking winners for backup
